# Project Mercenary

mer·cen·ary  
&nbsp;1. [noun] a professional soldier hired to serve in a foreign army  
&nbsp;2. [adjective] primarily concerned with making money at the expense of ethics    

## Stack
<b>Front</b> : Vue.js  
<b>Back</b> : Express.js  
<b>DB1</b> : MySQL (for basic info.)  
<b>DB2</b> : MongoDB (for chart)
